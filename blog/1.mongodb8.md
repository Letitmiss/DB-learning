


# MongoDB的启动,备份,锁,用户管理

## 启动项
|启动项| 描述 |
|----|----|
|--dbpath| 指定数据库的目录,windows默认是c:\data\db|
|--port|指定端口|
|--fork|用守护进程的方式启动mongodb|
|--logpath|指定用日志的输出路径,默认是控制台|
|--config|指定启动项的文件路径|
|--auth|用户认证方式启动数据库|

```
lfg1000708009:/home/mongdb/mongodb-linux-3.4.10/conf # vi mongod.conf 

port = 12345
dbpath = data
logpath = log/mongod.log
fork = true

```
